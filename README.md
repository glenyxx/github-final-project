# Simple Interest Calculator

A lightweight, automated Bash shell scripting application designed to compute simple interest profiles based on user inputs.

## Features
- Accepts input fields for **Principal amount**.
- Accepts input fields for **Annual rate of interest**.
- Accepts input fields for **Time period (in years)**.
- Dynamically outputs calculated Simple Interest directly to the terminal interface.

## Usage
Run the script using a Bash terminal:
```bash
bash simple-interest.sh
